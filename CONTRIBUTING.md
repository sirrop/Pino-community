# Contribution Guide
Pinoへのコントリビュート方法についてのガイドです。

## Issues
次のIssueを受け付けています。
- APIの使用に関する質問
- メソッドやクラスの問題の報告
- ドキュメントの問題の報告
- 改善の提案
- 新しい機能などの提案  

その他のIssueも歓迎しています

## Pull Request
Pull Requestはいつでも歓迎しています。
### 受け入れるPull Request
スペルミスや誤字修正のような細かなPull Requestは、Issueを立てずに送って構いません。  
次のようなものに関しては、Issueを立てて相談してください。
- ドキュメントの内容の修正
- メソッドの改善
- アプリケーションの改善
- テストの改善
